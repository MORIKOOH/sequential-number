## About

連番データを作るWebアプリ?です。

作成したデータはCSV形式やJSON形式でエクスポートできます。

[Github Pages](https://github.com/MORIKOOH/sequential-number/deployments/activity_log?environment=github-pages)

ファイルをエクスポートする際、ファイル名にタイムスタンプが付きます。
時刻の取得には moment.js を使用しています。
